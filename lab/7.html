1.1 Тема: ВИКЛЮЧНІ СИТУАЦІЇ.

1.2 Мета: придбати практичні навички роботи з виключними ситуаціями, які виникають при роботі програми.

1.3 Постановка задачі:

До програмного коду попередніх лабораторних робіт додати та описати:

- виключення, яке можуть виникати при некоректній авторизації

- виключення, з застосуванянм фільтрів виключення;

- виключення, які виникають при обробці масивів;

- системні виключення;

- власні виключення;

- протестувати програму. Зробити висновки.

1.4 Використання

Список виключень які реалізуються у даній лабораторній роботі з описом ціх виключень.

ArgumentException - вызывается при вызове метода и по крайней мере один из переданных аргументов не соответствует спецификации параметра вызываемого метода. Свойство определяет недопустимый аргумент.

АrgumentOutOfRangeException - сли значение аргумента находится вне диапазона допустимых значений.

InvalidCastException - Исключение возникает, если преобразование экземпляра одного типа в другой тип не поддерживается.

NullReferenceException - Исключение возникает при попытке получить доступ к члену для типа, значение которого равно null.

IndexOutOfRangeException - Исключение возникает, когда недопустимый индекс используется для доступа к члену массива или коллекции либо для чтения или записи из определенного расположения в буфере.

AuthException - кастомное, для определения ошибки аутентификации.

DivideByZeroException - Попытка деления целого числа или Decimal числа на ноль
